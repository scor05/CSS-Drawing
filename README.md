# CSS Only Drawing
---
## Descripción
Este reposotorio contiene un dibujo hecho para el laboratorio #4 de la clase de
Sistemas y Tecnologías Web. Traté (a lo mejor de mi habilidad hasta el momento)
de dibujar el logo del juego CS:GO (el antiguo, antes de que se volviera CS2), la
imagen de referencia siendo [la siguiente](https://cdn2.steamgriddb.com/file/sgdb-cdn/icon_thumb/767a40871fcc018629b26f3a9ff39e19.png).

Únicamente utilicé dos archivos para este repositorio: `index.html` y `styles.css`, los
cuales se explican por sí solos qué función tiene cada uno. El dibujo final se puede
visualizar en [`joelsiervas.online`](joelsiervas.online/24472/CSS-Drawing) o en [`cuchito.live`](cuchito.live/corderito/CSS-Drawing)

---
## Nota Importante
El dibujo se realizó con en el browser de Firefox Developer Edition,
por lo que pueden haber discrepancias con cómo se vea el dibujo con respecto al navegador que
se use para verlo. Por ende, para comprobar de que sí se ve correctamente el dibujo, favor compararlo
con la siguiente captura tomada en Firefox Dev:
[Imagen de backup](https://prnt.sc/aKr9j3Lez3ws)
